# v1.0.0 — 2025-08-05

Initial public release of the CLDMV org-level workflows.

## Added

- `publish-on-release.yml` — publish-on-release reusable.
- `release-please.yml` — initial release-please integration.
- `semrel.yml` — semantic-release scaffolding.

## v1.x minor series (v1.1 → v1.12)

The v1 line evolved across twelve minor releases and ~70 patch releases between this initial cut and the v2.0.0 transition — adding the action layer under `.github/actions/`, three-tier rolling tag strategy, GPG signing, dry-run mode, conventional-commit auto-detection, the unified tag-health pipeline, the org label sync, the coverage badge feature, the Node 24 runtime adoption (v1.12.13), and a large amount of workflow refactoring.

Minor changelogs for v1.1.0 → v1.12.0 don't have dedicated files yet. Each release is documented in its GitHub Release; they can be backdated on request.
